        <p>HTML is a basic structure language of websites. It controls the text, images, and "physical" blocks on a web page. It is also used to add links to other websites, documents. HTML is also useful for adding hidden elements with the help of CSS (Styling) for screen readers.</p>

        <ul>

          <li>Make hidden blocks and lists for organization</li>
          <li>Control the flow of screen reader technology</li>
          
        </ul>
        <p>CSS is a styling/markup language for websites. While it is mostly cosmetic, it provides most of the structure for a web page. It controls the colors, sizing, properties, and organization of HTML scripts.</p>
        <h2>CSS Can Be Used To:</h2>
          <ul>
            <li>Organize the flow and look of a web page</li>
            <li>Add color and shapes to HTML elements</li>
